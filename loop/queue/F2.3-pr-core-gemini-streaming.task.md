- Funções PURAS `gemini_body(model, system, user, max_tokens) -> serde_json::Value` e
  `gemini_extract(v: &Value) -> Result<String>` (unit-testáveis SEM rede), no formato da
  Gemini `generateContent`: corpo com `contents:[{role:"user",parts:[{text:...}]}]` +
  `system_instruction:{parts:[{text: system}]}` + `generationConfig:{maxOutputTokens}`;

  `gemini_body`, extrai via `gemini_extract`; `chat()` como os demais.

  `"gemini" => "gemini-2.0-flash"` ou o mais recente estável — escolha um default

  incerto — não inventar número preciso; `None`/estimativa conservadora é aceitável);

## Objetivo

Tanto a busca (`openHit`) quanto a navegação por xref empurram
`/read/[book]/[chapter]?verse=N`, mas a tela de capítulo lê só `{ book, chapter }` de
`useLocalSearchParams` (`app/app/read/[book]/[chapter].tsx:74`) e descarta `verse`
silenciosamente — então tocar um resultado de busca ou um xref de, ex., João 3:16 aterrissa
no TOPO de João 3. Ler o parâmetro opcional `verse` e, após a passagem renderizar, rolar o
`ReaderChapterView` (um `ScrollView` de `Pressable`s com `testID` `verse-${n}` e um estilo
`verseSelected` já existente) até esse versículo e aplicar uma seleção/destaque transitório.
Componente compartilhado → corrige web + nativo. 100% offline, sem tocar o `the-light`, sem segredo.

## Aceite

- [ ] Navegar para `/read/[book]/[chapter]?verse=N` rola o capítulo até o versículo N e aplica um
      destaque/seleção transitório a ele (reusando o conceito `verseSelected` e os testIDs `verse-${n}`).
- [ ] Abrir um resultado de busca ou um xref aterrissa no versículo alvo, não no topo do capítulo;
      quando não há parâmetro `verse`, a tela se comporta exatamente como hoje.
- [ ] `npx tsc --noEmit` 0; `npx expo export --platform web` sucede; qualquer cromo novo via `t()`
      com paridade pt/en (`npm run test:i18n-coverage` verde).
- [ ] Anti-alucinação: o texto do versículo continua vindo do store (só rola/destaca; não altera conteúdo).
      the-light intocado (`225b8c9`); sem segredo/rede. Sem regressão de `test:web:reading`/`xref`/`search`.

## Verificação

```sh
cd app && npx tsc --noEmit && npx expo export --platform web >/dev/null && echo EXPO_OK && npm run test:i18n-coverage && npm run test:web:reading && npm run test:web:xref
```

## Origem

`app/app/read/[book]/[chapter].tsx:74` (lê só book/chapter); `app/app/search/index.tsx:116-126`
(`openHit` empurra `?verse=N`); `app/components/ReaderChapterView.tsx:62-90` (ScrollView de
`verse-${n}`, estilo `verseSelected`). Varredura de refinamento (sweep code-gaps). Notas:
usar `ref`/`onLayout` + `scrollTo`/`scrollToIndex` conforme o padrão do componente; o destaque
transitório pode limpar após alguns segundos ou ao interagir — manter simples e determinístico.
Verificar que o xref também empurra `?verse=N` (se não, alinhar a origem de nav também, dentro do escopo).

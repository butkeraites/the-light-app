## Provas (anti-alucinação em todos os alvos — texto do store, nunca do LLM)
- **Nativo (iPhone 17, `simctl`):** `TLA_ASK ref="John 3:16" provider="mock" streamed=true
  cited_prefix="For God so loved the world..." interp_len=94` + validação REAL (F2.6):
  `cited_text` = João 3:16 KJV do store, interpretação real do Gemini.
- **Web (headless node, fetch MOCK):** `ai_web_prepare`→fetch→`ai_web_finalize` →
  `cited_text` = João 3:16 verbatim do store, `interpretation` da resposta; teste de host
  `ai_web_prepare_matches_native_anchored_cited_text_zero_drift` = **paridade Rust**.
- Sem regressão da Fase 1 (READ/PARALLEL/SEARCH/XREF/NOTES nativos + reading/search/xref/
  notes web todos verdes).

## Regras não negociáveis — TODAS mantidas
- **BYOK / chave nunca em git/log:** nativa em Keychain/Keystore (F2.4); web session-only
  in-memory (ADR-0025). **Login-de-conta OAuth REJEITADO** por pesquisa (banido/indisponível
  nos 3 provedores — ADR-0023); só API key oficial.
- **Anti-alucinação:** `cited_text` sempre do store local (verbatim); o LLM só interpreta;
  `AiAnswer` separa os dois; `citation::rewrite_anchors` (mesma impl Rust nativo↔web) —
  **zero drift**.
- **Offline-first:** IA opt-in; a Fase 1 segue 100% offline; rede só no `fetch`/HTTP com a
  chave do usuário.
- **`the-light` só via PR+ADR:** 2 PRs sancionados (F2.3 Gemini+streaming `133077a`; F2.7
  `ai-pure`+fix modelo `c8ecb2f`), cada um mergeado por sign-off humano + re-pin; nunca
  forkado. Grafo wasm permanece puro (sem rusqlite/reqwest).
- **Qualidade por tarefa** verde; cada tarefa revisada de forma independente.

## Decisões de arquitetura (ADRs desta fase)
ADR-0023 (gate F2.2: Gemini+web via PR ao core; **BYOK API key, OAuth rejeitado**;
streaming) · ADR-0024 (feature `ai-pure` wasm-safe + fix `default_model` gemini→2.5-flash) ·
ADR-0025 (chave web session-only + padrão prepare/fetch/finalize).

## Fora de escopo (registrado, não é regressão)
- **Streaming web real** (token-a-token via fetch stream): a IA web é não-streaming por ora
  (o nativo tem streaming via `complete_stream` default; SSE real por provedor = evolução).
- **`*_body`/`*_extract` do core `pub`** (para o body/extract do provedor virem do Rust
  também no web): hoje o transporte web é TS (infra, ADR-0025); torná-los `pub` = PR futuro.
- **Corpus completo web (~59 MB)** = backlog transversal (Fase 1).
- **Fase 3+** (estudo profundo: léxico Strong/pesquisa; o core já tem `ai::study`/`lexicon`/
  `research` — embedded) — a decompor após o sign-off.

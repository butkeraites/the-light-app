## Resumo

F3.7 entrega a **UI nativa de comparação multi-IA** (`ReaderComparePanel`) ancorada numa
passagem: uma pergunta + seletor MULTI de provedores (`[...SUPPORTED_PROVIDERS, 'mock']`, ≥2)
que dispara **N chamadas independentes** a `askAnchored` (F2.1/F2.3a, reusada — sem função de
fronteira nem glue novo) sobre a **mesma** `reference`, via `Promise.all`. O `citedText`
(âncora, verbatim do store) é IDÊNTICO em todas as colunas → exibido **UMA vez** como bloco
comum "Passagem (texto bíblico) — âncora comum", **separado** das N `interpretation` (LLM)
rotuladas por `provider · model` (anti-alucinação visível). Acessível via botão "Comparar
(IA)" (`testID="verse-compare"`) no `ReaderVersePanel`, montado no `[chapter].tsx`. Prova
determinística por MOCK no device (`TLA_COMPARE`) + web SKIP. Verifiquei de forma
independente: `tsc`/`expo export web` verdes, self-test iOS exit 0 com `TLA_COMPARE` e sem
regressão dos demais `TLA_*`, greps de anti-alucinação/anti-leak/wiring OK, `the-light` e o
core Rust intocados. Decisão: **passed**.

## Decisões / ADRs

- **Sem ADR novo.** F3.7 é UI + self-test + wiring; reusa a fronteira `ask_anchored`
  (F2.1/F2.3a) e o glue `askAnchored` (`reading.ts`/`reading.web.ts`) já existentes — nenhuma
  função/tipo novo de fronteira, nenhum glue novo, sem PR ao `the-light`.
- **Desenho da comparação:** N chamadas independentes a `ask_anchored` (uma por provedor)
  sobre a MESMA `reference`; o `citedText` (âncora, verbatim do store) é comum e exibido UMA
  vez, separado das N `interpretation`. O invariante `cited_match` (todos os `citedText`
  iguais e não-vazios) é a anti-alucinação + prova de que as colunas leram a mesma passagem.
- **Prova por MOCK = WIRING de N provedores**, não comparação de conteúdo: o `MockLlmProvider`
  é fixo, logo mock×mock é degenerado. A comparação REAL de respostas (Claude/GPT/Gemini,
  chave real) é a **F3.10** (gate). Registrado no código e no self-test.
- **BYOK/offline-first:** chave de provedor real viria de `getKey(provider)` (keystore F2.4) e
  a prova NÃO a toca (só `"mock"`, sem chave/rede); chave nunca logada/exibida. Custo omitido
  (fronteira não expõe `estimate_cost_usd`, idem F2.5).

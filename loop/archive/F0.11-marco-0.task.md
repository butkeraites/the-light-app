## Objetivo

**Gate de Marco 0.** Ponto de decisão humana/auditoria que fecha a Fase 0 (prova de
viabilidade da ponte Rust→Expo). Por ser `gate: true`, o loop **NÃO** executa esta
tarefa autonomamente: o Driver escreve `loop/HALT` e o **humano/Guia** revisa e dá o
sign-off antes de liberar a Fase 1.

## Critério de revisão (checklist do Marco 0)

- [x] **Ponte UniFFI→Expo provada nos 3 alvos** com run real:
  - Web/WASM (F0.6b, `1cdde6c`) — `parseReference` no navegador.
  - iOS (F0.7, `a6f6797`) — Turbo Module no simulador.
  - Android (F0.8, `36af016`) — Turbo Module no emulador.
- [x] **Fricção #1 (SQLite no WASM)** resolvida: feature `embedded` no `the-light-core`
  (PR mesclado, `8f66004`) + matriz de features por alvo (ADR-0005).
- [x] **Store provado** nativo (F0.9, `1d16897`, `rusqlite`/core) e web (F0.10,
  `d6e968d`, `wa-sqlite`+OPFS, ADR-0011/0012).
- [x] **Regras não negociáveis respeitadas:** `the-light` consumido sem fork (git dep
  pinada; única mudança via PR+ADR sancionado); anti-alucinação (texto sempre do store
  local, verbatim domínio público); offline-first (rede só em dev/build); BYOK n/a ainda;
  segredos nunca em git.
- [x] **ADRs e PROGRESS atualizados** (ADR-0001..0012; `PROGRESS.md` com F0.0–F0.10).

## Decisão pedida ao humano/Guia

1. **Aprovar o Marco 0** (viabilidade da Fase 0 comprovada) e liberar a **Fase 1**
   (leitura offline multiplataforma — ver `IMPLEMENTATION_PLAN.md` §F1), **ou**
   apontar ajustes antes de seguir.
2. Confirmar se há itens de "dívida"/follow-up da Fase 0 a registrar como tarefas
   (ex.: bundling do `sample.sqlite` no app nativo iOS/Android para o `get_passage`
   nativo via Turbo Module; store web completo — search/xref — via Opção B no futuro;
   limpeza do AVD/symlink; CI dos scripts de prova headless).

## Como liberar

Humano/Guia revisa o estado (`STATUS.md`, `PROGRESS.md`, `loop/archive/*.result.md`,
`DECISIONS.md`, git log), e então **remove/edita `loop/HALT`** com o veredito. Se
aprovado, o planner decompõe a **Fase 1** no `backlog/`/`queue/` e o loop retoma.

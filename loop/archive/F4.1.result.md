## Resumo

Streaming WEB real da IA entregue **só no transporte TS**, com ZERO DRIFT verificado
de forma independente. `geminiCompleteStream` lê `res.body.getReader()` + `TextDecoder`,
parseia eventos SSE `data:{…}` (endpoint `:streamGenerateContent?alt=sse`), extrai cada
DELTA (`geminiPartText`, mesmo shape do core), chama `onToken(delta)` na ORDEM e ACUMULA
o texto completo. `webLlmTransport(...,onToken?)` despacha `"gemini"`→stream e `"mock"`→
`emitMockStream` (offline). `askAnchoredOnHandle(...,onToken?)` repassa o callback; a
`interpretation` acumulada vai à MESMA `aiWebFinalize`. `reading.web.ts::askAnchoredStream`
destubado (assinatura pública e `AiAnswer` final inalterados). A fronteira Rust `ai-pure`
(`ai_web_prepare`/`ai_web_finalize`) NÃO muda; `the-light` intacto (`04b9b24`); `core/**`

nos 3 cenários; tsc + expo export web verdes; 12 testes web PASS (sem regressão).

**Linha-chave:** streaming: concat(tokens) == `AiAnswer.interpretation` (6× na ordem) +
âncora espúria `[V:G9999]` REMOVIDA pela MESMA `ai_web_finalize` (Rust `rewrite_anchors`)

## Critério de aceite

- [x] **N ≥ 2 chunks SSE → `onToken` N× (não 1×) + concat == `interpretation`.** Cenário A:
      6 deltas → `onToken` 6×, `tokensA.length == deltasA.length`, `deepEqual(tokensA,
      deltasA)` (ORDEM) e `tokensA.join('') == answerA.interpretation` (`"This passage
      speaks of God's love."`). N≥2 asserido.
- [x] **`citedText` == `"16 <João 3:16 KJV VERBATIM>"` via `ai_web_prepare`, separado da
      interpretação.** Asserido em A/B/C; `!interpretation.includes('For God so loved')` e
      `notEqual(citedText, interpretation)` — anti-alucinação (NÃO vem do mock/stream).
- [x] **Âncora Strong espúria `[V:G9999]` REMOVIDA por `ai_web_finalize` (Rust).** Cenário
      B (bytes fatiados 7B, eventos SSE quebrados entre chunks): stream contém `[V:G9999]`
      (`rawB.includes`), mas `!answerB.interpretation.includes('[V:G9999]')` e
      `interpretation == rawB.replaceAll('[V:G9999]','')` (só a âncora removida, nada mais).
- [x] **1 `fetch`, endpoint de streaming POST, chave só no header.** `callsA.length == 1`;
      URL começa em `…/models/gemini-2.5-flash:streamGenerateContent`, inclui `alt=sse`,
      `!url.includes(DUMMY_KEY)`; `method == 'POST'`; `headers['x-goog-api-key'] == key`.
- [x] **`"mock"` + `onToken` emite ≥1 incremento OFFLINE (0 fetch); não-streaming sem
      regressão.** Cenário C: `callsC.length == 0`, `tokensC.length == 15 (≥1)`,
      `concat == interpretation`. `test:web:ai` (não-streaming, F2.7b) PASS.
- [x] **`core/src/lib.rs`/`core/Cargo.toml` inalterados; `the-light` `04b9b24` intacto;

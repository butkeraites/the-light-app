## Resumo

F2.1 abre a **Fase 2 (IA BYOK)** expondo `ask_anchored(...)` na fronteira UniFFI
(`core/src/lib.rs`), que **delega** integralmente à camada de IA do `the-light-core`
(`ai::{build_provider, numbered_passage, ask_context, ask}`) e prova a espinha ponta a
ponta com o **provedor MOCK** (`build_provider("mock", None, None)` — sem chave, sem
rede). O Record `AiAnswer{reference, cited_text, interpretation, provider, model}`
**separa** o `cited_text` (passagem numerada **verbatim do store**, via
`ai::numbered_passage` sobre a `Passage` lida por `EmbeddedSource::passage` — mesma rota
da F1.2) da `interpretation` (saída do modelo, via `ai::ask`) — a anti-alucinação
materializada no contrato. A fronteira **não reimplementa** prompt/RAG/roteamento de
provider: só canonicaliza a referência (`parse_reference`/`format_reference`), lê o store
e chama o `ai`. Corpo `cfg(not(wasm32))` + stub web (`"ai não disponível no alvo web
(F2.7)"`), Record puro montado à mão (sem `From<ai::…>`). Verificação independente:
`fmt`/`clippy -D warnings` limpos, **37/37** testes verdes (3 novos `ai_tests` + 34
prévios), grafo wasm **puro** (sem `rusqlite`/`reqwest`), `gen-bindings-web.sh` exit 0
com `askAnchored`/`AiAnswer` no web. `the-light` intocado (`8f66004`), `core/Cargo.toml`
inalterado. **passed.**

## Decisões / ADRs

- **Sem ADR novo.** F2.1 segue os ADRs já vigentes: o módulo `ai` do core é
  `#[cfg(feature = "embedded")]` (só nativo, ADR-0005), então `ask_anchored` tem corpo
  `cfg(not(wasm32))` + stub web (gating de corpo, ADR-0010) — o grafo wasm permanece puro
  (sem `reqwest`/`rusqlite`; paridade web de IA = F2.7).
- **Contrato anti-alucinação (SPEC §6.2):** `AiAnswer` separa `cited_text` (numerado,
  **verbatim do store**, `ai::numbered_passage`) de `interpretation` (saída do LLM,
  `ai::ask`). O Record é **puro** e montado à mão (não há tipo-fonte único do core a
  converter → sem `From<ai::…>` a gatear, diferente de `CrossRef`/`Note`).
- **BYOK:** a chave é **argumento** de `build_provider` (não `secrets.toml`/`KeyStore`);
  esta tarefa usa `key = None` + provider `"mock"` (sem rede/chave). Armazenamento seguro
  real = F2.4; rede real com chave do usuário = F2.6 (bloqueante).
- **Arquitetura estratégica de IA** (onde vive o Gemini, IA no web, chave web, streaming,
  ADR-0023) permanece **fora de escopo** aqui — é o gate **F2.2**.

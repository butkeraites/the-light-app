## Decisões / ADRs

- **ADR-0021** (aceito, registrado no commit): Xref WEB (A1) — espelho TS do SELECT de
  `xref::for_verse` (votos DESC + tiebreakers + `min_votes` + `LIMIT` clamp ≥1 +
  Single/Range) sobre o subset `reading-sample.sqlite`, REUSANDO `openReadingDbWeb`
  (MESMO wasm/store das F1.13/F1.14, sem recarregar o subset), destubando `crossRefs`.
  Xref é independente de tradução → sem `translation`/`has_translation` (≠
  `getChapter`/`search`). Tipo `votes` i64 → **bigint** no binding (`column_int64` na
  leitura, `bind_int64` em `min_votes`/`limit`), fiel ao contrato do core.
- **ADR-0016** (atribuição CC-BY) visível no web: string EXATA `Cross references courtesy
  of OpenBible.info (CC-BY)` no componente compartilhado, confirmada no bundle
  `expo export --platform web`.
- ADR-0018/0019/0020 reusados: subset A1 + store web de leitura/busca (wa-sqlite vendored,
  MemoryVFS/OPFS) sem novo backend.
